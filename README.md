# Customer Churn Prediction

## Sobre o Projeto
Este projeto tem como objetivo prever a evasão (churn) de clientes em um serviço por assinatura, identificando quais usuários possuem maior probabilidade de cancelar antes que isso aconteça. O foco é construir um pipeline de Machine Learning aplicado a um problema real de negócio, lidando com dados desbalanceados e extraindo insights acionáveis. Além do desenvolvimento do modelo, o projeto busca interpretar os resultados de forma clara, permitindo que áreas não técnicas (como marketing ou retenção) utilizem as previsões para tomada de decisão.

## Objetivos
- Prever churn de clientes
- Trabalhar com dados desbalanceados
- Treinar e avaliar modelos de classificação
- Gerar insights para retenção de clientes

## Dataset
- Telco Customer Churn (Kaggle)
- Dados de clientes de telecomunicações
- Variável alvo: churn (sim/não)

## Tecnologias
- Python
- Pandas
- Scikit-learn
- Imbalanced-learn

## Modelos Utilizados
- Logistic Regression

## Avaliação
- Accuracy
- Precision
- Recall
- F1-score
